# DSL to C Code Generator

Тестовое задание — генератор C-кода из XML-описания блок-схем.

**Что реализовано:**
- Парсинг XML (pugixml)
- Топологическая сортировка блоков
- Генерация `nwocg_generated.c` + `.h`

Добавьте xml файл `PI_Controller.xml` в папку проекта для работы с этим xml
или укажите полный путь в main() внутри testTask.cpp

Результат соответствует стилю примера из тестового задания.
